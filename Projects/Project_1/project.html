Power BI Spotify Analytics Dashboard
Oct 17, 2024

Project Overview:

This interactive Power BI dashboard provides in-depth insights into Spotify streaming analytics and track performance from 2017 to 2023. It visualizes music streaming patterns, track performance, and helps users understand listening behavior over time.

Why I Built This:

In a data-driven world, businesses and individuals need to make sense of large volumes of data. With services like Spotify, this dashboard uncovers valuable insights from listening patterns and track performance.

  • Help music industry professionals (artists, managers, record labels) better understand audience preferences.
  • Enable data enthusiasts to analyze trends in music consumption.
  • Provide interactive and visually compelling insights for exploring streaming data effectively.

What I Did:

  • Developed dynamic data visualizations to analyze music streaming trends and metrics.
  • Implemented an interactive filtering system for exploring specific time frames, artists, or songs.
  • Integrated Spotify API data to fetch personal streaming history and track metadata.
  • Designed a user-friendly dashboard with custom visual elements like bar charts, heat maps, timelines, and circular gauges.

Key Features:

1. Streaming Analytics:

  • Visualize average streams per year (417.81M with 787% growth).
  • Historical streaming trends (2017-2023).
  • Monthly streaming patterns and growth rates.

2. Track Performance Analysis:

  • Track scoring based on:
    • Danceability, Liveness, Acousticness, Valence, Speechiness.
  • Energy level measurement.
  • Top tracks by stream count.

3. Interactive Data Exploration:

  • Filters for refining data by year, artist, track, or custom date range.
  • Compare trends across different time frames or artists.

4. Data Visualizations:

  • Bar charts for stream comparisons.
  • Heat maps to display monthly and yearly streaming patterns.
  • Timelines for track release dates and streaming trends.
  • Circular gauges for energy level visualization.
  • Performance matrices for side-by-side track attribute analysis.

How It Benefits Clients:

  • Music industry professionals can identify high-performing tracks and make informed marketing and promotional decisions.
  • Artists can analyze which songs resonate with their audience to guide future music production.
  • Data analysts can uncover trends in music consumption, predicting future behaviors or rising genres.

Technologies Used:

  • Microsoft Power BI for creating the dashboard and visualizations.
  • DAX (Data Analysis Expressions) for complex calculations.
  • Power Query Editor for data transformation.
  • Spotify API for fetching streaming data and track metadata.

How To Use This Dashboard:

  1. Clone the repository: 
    git clone https://github.com/Amgtps/PowerBI-spotify.git
cd PowerBI-spotify
  2. Open the .pbix file in Power BI Desktop.
  3. Connect your Spotify account to load personal data or explore using the provided sample data.
  4. Refresh the data to start analyzing your Spotify streaming metrics.
